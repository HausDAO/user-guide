# DAOhaus User Guide

The official user guide for [DAOhaus](https://daohaus.club/), published at [guide.daohaus.club](https://guide.daohaus.club/).

This guide covers how to summon and operate Moloch v3 DAOs using the DAOhaus [Admin App](https://admin.daohaus.club/). It was built by the PublicHAUS community.

## Contents

- **How To** — Step-by-step guides for summoning a DAO, navigating the Admin App, managing proposals, members, safes, and settings
- **Quickstart** — Focused walkthroughs for common tasks: joining as a member, transferring tokens, migrating from v2
- **Glossary** — Definitions for Moloch DAO terminology
- **Community Links** — Discord, GitHub, X, and Mirror

## Stack

Built with [Nextra](https://nextra.site/) and Next.js. Pages are written in MDX.
